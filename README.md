# Sequential Circuits in Verilog

This repository contains Verilog implementations of fundamental sequential circuits used in digital design. These circuits store information and change state based on clock inputs or triggers.

## 🧠 Contents

- SR Latch
- SR Flip-Flop 
- **D Latch**
- **D Flip-Flop**
- **T Flip-Flop**
- **JK Flip-Flop**
- **Counters**
  - 2-bit / 4-bit Up Counter
  - Down Counter
  - Up-Down Counter
- **Shift Registers**
  - Serial-In Serial-Out (SISO)
  - Serial-In Parallel-Out (SIPO)
  - Parallel-In Serial-Out (PISO)
  - Parallel-In Parallel-Out (PIPO)

## 📁 Folder Structure

Each module includes:
- Verilog source code
- Testbench
  
Feel free to contribute or use these modules in your projects!
